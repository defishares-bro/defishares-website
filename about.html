About DefiShares

What is DefiShares? DFS asset network overview

DefiShares is an independent blockchain network forked from the BitShares / Graphene technology stack. It is designed as an official access layer for DFS, GOLD-centered smart asset policy, wallet access, explorer lookup, airdrop information, and public project documentation.

DefiShares provides five general functions for the public interest of its ecosystem:

  • maintains DFS as the core asset for fees, accounts, transfers, and network operations;
  • uses GOLD as the protocol pricing anchor for DefiShares-managed smart assets;
  • provides official wallet and explorer access for users to interact with and verify chain activity;
  • publishes documentation for chain parameters, risk policy, airdrop status, and asset mechanisms; and
  • supports community governance, witness operations, and transparent ecosystem communication.

Governance and Operations

DefiShares follows a witness-based network model inherited from the Graphene family. Witnesses maintain block production and participate in feed publication for applicable assets. Public documentation should distinguish verified project links, network state, asset policy, and risk disclosures from community commentary.
